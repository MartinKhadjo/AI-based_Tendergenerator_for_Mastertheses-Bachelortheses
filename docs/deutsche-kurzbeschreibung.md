# Deutsche Kurzbeschreibung

Der **AI-based Tendergenerator für Bachelor- und Masterarbeiten** automatisiert die Erstellung professioneller PowerPoint-Ausschreibungen. Über eine Browseroberfläche werden Titel, kurze Beschreibungstexte, Kontaktname, Sprache und optional ein Bildordner erfasst. Das System pseudonymisiert personenbezogene Angaben, erweitert die Inhalte KI-gestützt, wählt die passende Vorlage und erzeugt eine fertige PPTX-Datei.

Dieses öffentliche Showcase-Repository dokumentiert Architektur, Workflows und Diagramme, ohne den proprietären Quellcode offenzulegen.
