# Engineering Boundaries

This repository is intentionally a **showcase repository**, not a public source repository.

## Included
- product and workflow explanation,
- architecture and diagram evidence,
- deployment and privacy documentation,
- release/provenance guidance,
- recruiter-facing portfolio material.

## Excluded
- proprietary source code,
- private prompt design,
- internal templates and mappings,
- configuration secrets,
- internal institutional data,
- any material that would enable trivial reconstruction of the product.

## Rationale
The goal is to communicate engineering quality and project scope while protecting intellectual property and respecting security/privacy boundaries.
